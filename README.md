# 문제 인식
### 결식 아동이란?
결식아동은 빈곤으로 인해 결식할 위험이 있는 한부모, 소년소녀가장, 
저소득층(중위소득의 52%미만)가정의 아동을 뜻한다

### 아동 급식 카드
경제적 빈곤 상태에 놓여 있는 가정의 자녀가 학교에서 급식을 먹지 못할 경우 학교 바깥에서 급식에 준하는 식사를 할 수 있도록 정부에서 보조하는 카드이다.
